RAÍCES v102 — MULTIPLE DELIVERY ADDRESSES

Before testing addresses:
1. Open Supabase > SQL Editor.
2. Copy and run: supabase/customer_addresses.sql
3. Deploy this version through GitHub/Netlify.
4. Log in and test My Account > Addresses.

Included:
- Add multiple addresses
- Edit and delete addresses
- Set one default address
- Spanish/English interface
- Data protected by Supabase Row Level Security

Google address autocomplete is not activated in this version because it requires a Google Cloud API key restricted to myraices.com and billing enabled.
