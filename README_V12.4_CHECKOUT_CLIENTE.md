# MyRaices v12.4 — Checkout orientado al cliente

Cambios incluidos:

- El botón ahora muestra “Continuar al pago seguro”.
- Se eliminaron del checkout los textos técnicos visibles sobre Square Sandbox y la versión interna.
- El mensaje bajo el resumen explica de forma simple que el pedido y el delivery se verificarán antes del pago.
- En pedidos compuestos únicamente por productos digitales se ocultan teléfono, dirección, apartamento, ciudad, estado, ZIP e instrucciones de entrega.
- Google Maps no se carga cuando el carrito contiene solamente productos digitales.
- Para productos digitales solo se solicitan nombre y correo electrónico.
- Se mantiene intacta la lógica existente de Square, Supabase, inventario y compatibilidad con NURAI.
