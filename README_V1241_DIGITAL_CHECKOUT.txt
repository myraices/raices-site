RAÍCES v12.4.1 — Checkout digital

- Carrito exclusivamente digital: muestra únicamente Nombre y Email.
- Usuario autenticado: Nombre y Email se completan desde Supabase Auth.
- Usuario no autenticado: puede escribir únicamente Nombre y Email.
- Carrito físico o mixto: muestra y exige teléfono y dirección completa verificada.
- El request de un carrito digital envía únicamente nombre y correo como datos del cliente.
- Corrige el conflicto CSS que impedía ocultar los campos de entrega.
